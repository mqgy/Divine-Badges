Divine Badges is a customizable Minecraft badge system plugin designed for servers that include gym leader style gameplay mods (Cobblemon, Pixelmon, etc.).

Admins can create custom badges, assign gym leaders and track player progress through an interactive GUI.
Players can view their earned badges while gym leaders can award badges through a simple menu.

Features:

- Custom badge creation
- Ability to assign gym leaders to badges
- Interactive badge GUI
- Award badges through menu and command
- Badge progress tracking
- Pages for large badge collections
- Sound effects when receiving badges
- Visual indicator for earned badges

Commands:

Player Commands:
- /badges	Opens your badge collection
- /badge award	Opens the award menu (gym leaders only)

Admin Commands:
- /badge create <id> <material> <name>	Create a badge
- /badge delete <id>	Delete a badge
- /badge list	List all badges
- /badge give <player> <badge>	Force give badge
- /badge revoke <player> <badge>	Remove badge from player
- /badge setleader <player> <badge>	Set a gym leader
- /badge removeleader <player> <badge>	Remove a gym leader
- /badge reload	Reload badge data
